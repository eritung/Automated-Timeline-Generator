# Automated Timeline Generator｜製作時程排程工具

這是一個使用 Streamlit 製作的製作時程排程工具，可依專案開始日、上線日與工作流程，自動產出可下載的 Excel 時程表。適合用於廣告素材、網頁製作、專案排程等需要快速整理工作天數與確認節點的情境。

## 使用前提醒：Streamlit Wake Up

本專案目前部署於 Streamlit 上。若使用 Streamlit 線上版操作，不需要在本機安裝 Python，也不需要另外安裝套件；只要開啟 App 頁面即可使用。

由於 Streamlit App 長時間未使用後可能會進入休眠狀態，因此每天第一次使用前，請先進入 App 頁面並點擊 **Wake up**，讓系統重新啟動服務。

若畫面顯示 App 正在喚醒中，請等待頁面載入完成後再開始操作。若載入後畫面沒有正常更新，可重新整理頁面一次。

## 主要功能

- 支援依「製作日推進」、「上線日回推」或「同時指定開始與上線日期」產出時程。
- 可自訂專案名稱、開始日期、上線日期與日期縮略門檻。
- 可設定假日清單，並選擇上線日是否可排在國定假日。
- 支援單筆編輯任務，可新增、複製、刪除、排序與修改工作天數。
- 支援批次輸入流程，快速套用一般製作時程或網頁製作時程範本。
- 支援 0.5 天工作項目，並可自訂半天標註文字。
- 可匯入本工具產出的 Excel 時程表，回填任務名稱、Action By 與工作天數。
- 可下載 Excel 格式時程表，方便後續提供客戶或內部團隊使用。

## 本機安裝方式（僅本機執行需要）

若只使用 Streamlit 線上版，請略過本段，不需要在本機安裝 Python。

若需要在自己的電腦上執行此專案，請先確認已安裝 Python，接著在專案資料夾中執行：

```bash
pip install -r requirements.txt
```

## 本機啟動方式

在專案資料夾中執行：

```bash
streamlit run app.py
```

若要啟動另一版視覺樣式，可執行：

```bash
streamlit run app_glassmorphism.py
```

## 基本操作流程

1. 進入 Streamlit App；若 App 已休眠，請先點擊 **Wake up**。
2. 使用 Streamlit 線上版時，可直接操作，不需在本機安裝 Python。
3. 於「專案設定」輸入專案名稱，並選擇排程方式。
4. 設定開始日期或上線日期。
5. 依需求調整假日設定與上線日規則。
6. 於「單筆編輯」或「批次輸入」設定工作流程。
7. 點擊「產出時程表」查看排程預覽。
8. 確認無誤後點擊「下載 Excel」。

## 批次輸入格式

每行一筆任務，使用空白區隔資訊：

```text
任務名稱 ActionBy 工作天數 [0.5文字] [上線]
```

範例：

```text
提供素材 客戶 1天
視覺製作 Ad2 3天
客戶確認 客戶 0.5天 1300
廣告上線 Ad2 1天 上線
```

若要在上一筆任務下方加入粗分隔線，可單獨輸入：

```text
--
```

## 假日設定格式

假日清單每行一筆，格式如下：

```text
YYYY-MM-DD,假日名稱
```

範例：

```text
2026-01-01,元旦
2026-02-16,春節連假
2026-02-17,春節連假
```

## Excel 匯入說明

可上傳由本工具下載的時程表 Excel，系統會自動讀取並帶回任務名稱、Action By 與工作天數。若原 Excel 中有手動調整色條，也會盡量納入天數計算。

匯入後請點擊「匯入時程表並套用到流程」，不需要再另外點擊「套用手動批次內容」。

## 注意事項

- 建議使用本工具產出的 Excel 進行匯入，避免格式不同造成讀取失敗。
- 若調整半天項目的標註文字，請重新產出時程表再下載。
- 若 Streamlit 頁面長時間沒有反應，可重新整理頁面，或重新點擊 Wake up。
